简单好用强大的个人文库管理工具——xSidebar

Created2021-05-16Updated2021-05-16

软件说明

此工具主要配合 markdown 文本编辑器使用，如 typora（强烈推荐），能够帮助你整理、检索（正则搜索很厉害）和分享个人文库中的内容。如果是搞科研的童鞋，再搭配 Mendeley（其他没有测试过，但理论上 bib 文件对于 zotero、jabref 都会很方便），做文献阅读笔记和写论文文草稿也十分方便，因为本软件支持导出带参考文献的 word 文档，以及导出 self-contained HTML （离线独立网页，可嵌入图片，音乐以及视频）而且还能够收集你在 markdown 文稿中插入的参考文献形成新的文献库子集。

安装方式

下载文件解压后双击 exe 即可使用。

百度网盘链接：https://pan.baidu.com/s/1LTuK1ffHfXWII9oN3qJHug

提取码：v273

使用方法

快捷键

  • Enter：搜索关键词
  • Ctrl+N：新建草稿
  • Ctrl+R：重设软件配置参数（Reset）

关键词搜索

  • 仅支持多关键词 AND 搜索（精确匹配模式），多个词用空格分开即可
  • 模糊匹配建议使用正则表达式进行搜索，如正则.{1,3}可以同时匹配“正则式”、“正则模块”与“正则表达式”三个关键词，详见菜鸟教程
  • 搜索结果返回匹配文本的文件名，并在列表中显示
  • 点击搜索结果可以在消息框中预览文件 Front Matter
  • 当输入字符少于两个时，返回当前目录下的文件列表

文稿列表的操作

  1. 鼠标左键单击显示文稿的 Front Matter
  2. 右键召唤出菜单，执行各种针对文稿的操作
  3. preview 生成临时网页用浏览器预览
  4. open 调用 typora 编辑器打开文稿
  5. asset 打开文稿对应的附录文件夹
  6. toHTML 导出 self-contained 的 HTML 网页文件到桌面
  7. toDocx 导出带有参考文献 bibliography 的 docx 文件到桌面
  8. toBib 收集并导出参考文献条目到 refs.bib 并存放到附录文件夹
  9. cite 引用被选中的文稿，然后自动复制 :link:《[title](filename.md)》 到粘贴板中，方便快捷插入对文稿的引用
  10. 最后是插入附件的按钮，你可以点击按钮，选择对应的图片、音乐、视频和其它任意文件作为附属资源插入被选中的文稿中，程序会把文件拷贝到附录下，然后生成相对路径的引用文本到粘贴板，接下来只需要到 typora 粘贴即可

其他重要设定

  • Open Folder 可以切换到其它文稿目录，要回到 config.yaml 设定的 postdir，可以点 reset 按钮。
  • 为避免报错，导出 HTML 和 Word 都建议在联网条件下进行，详见常见问题2

文章草稿的Front Matter

可以将文稿存放目录指定为 hexo 站点的 _posts 目录，生成文稿符合 hexo 文章要求，具有 Front Matter，同时已经适配了 hexo 的 stun 主题，所以 Front Matter 默认具有以下内容：

  • title：标题，不能为空，若创建时不写标题，则以当天日期为题，如20210429
  • excerpt：摘要，不能为空，使用提示作为占位
  • date：时间戳，不能为空，自动生成
  • tags：文稿的标签，自由定义，最好不超过五个
  • categories：文稿的分类，默认分类为草稿

其它问题

GUI界面缩放异常

对于一些分辨率很高的屏幕，系统缩放导致界面显示异常，这个时候需要对 python.exe 进行兼容性设置（右键属性》兼容性》更改高DPI设置》替代高DPI缩放行为》系统增强）。

本程序在 15.6 英寸笔记本电脑屏幕，125%的推荐缩放下取得最好的显示效果。

无法导出 html 和 docx

导出带参考文献的 docx 根据所使用的 csl 样式文件规定，可能需要联网。导出嵌入图片的 html 文件时，如果图片是指向网络图片而非本地附录文件夹中的图片，则需要联网。
