文献精读工作流

Created2021-04-25Updated2021-04-25

很多时候要针对一个主题进行文献阅读，会关注不同文章的不同局部细节内容，并且做笔记。做完笔记后，可以把相关的参考文献重新汇总一下，和阅读笔记一起放到 mendeley 的新建子目录内。这个过程涉及到一些信息的提取和文件格式的转化，因此整合了一些代码，方便自己。

具体方法是这样的：

  1. 使用newPost.bat脚本在_posts目录下生成一个带有 front matter 模板的 markdown 文件
  2. 打开 mendeley 查找和阅读刚兴趣主题的文献，并在 markdown 中做笔记
  3. 修改和补充 front matter 中的信息，title、excerpt、tags等信息是必需的
  4. 从typora中直接复制 markdown 的文件路径，作为修改上面collectCitations.py 中 md 的值（字符串），然后运行脚本
  5. 生成的 refs.bib 文件保存在 markdown 笔记文件的附录文件夹中，可以从 mendeley 中新建一个目录，然后 import 这个 bib 文件
  6. 将 markdown 笔记用 typora 导出为 html，然后用 htmlEmbedding.py 脚本导出为 standalone 的网页，作为附件添加到 mendeley 相应笔记条目的 files 中
