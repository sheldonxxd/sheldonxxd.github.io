单指数分布数据的作图

Created2021-04-12Updated2021-04-12

从博士到现在，originLab 用的越来越少， python 用得越来越多，不仅仅是数据分析，作图也是如此，这篇文章简单记录了绘制单指数直方分布图表的搞法。

作图效果

先放效果图：

左侧是一个单指数下降的直方分布（关于 τon ），右侧是一个累积分布（关于 τoff ），数据点我就用圆圈 marker 来表示。然后曲线是单指数拟合（y=a∗(1−np.exp(−x/b))+cy=a*(1-np.exp(-x/b))+cy=a∗(1−np.exp(−x/b))+c）的曲线，而虚线是根据拟合函数的参数，计算出来的 τ 时间。

读取数据

这一段代码可圈可点：

  1. 使用os.path.join组合路径，然后使用*通配符，借助 glob 函数批量读取带有特定后缀的文件，返回符合特征的文件路径列表，这种处理文件路径的方式比较轻松。不过 glob 读文件的列表顺序并不是按照命名来的，这个要注意下。
  2. hdf5 文件是常见的数据文件，python 中有 h5py模块来完成对此类文件的读取和写入，我这里为了方便，直接使用 pandas 模块中提供的方法来读取数据，适用于数据文件内容为表格或者矩阵类型的情况。
  3. 原始表格比较大，很多列的信息其实这里并没有被用到，所以我使用 pd.DataFrame 对象所拥有的 filter方法实现对数据列的过滤。

作图效果如下：

这里就不作赘述，只提一下plt.plot中的几个参数：

  1. mfc：marker face color，可以理解为 marker 的填充颜色
  2. ‘o’：定义圆圈作为 marker
  3. mec： marker 的 edge color，边缘颜色
  4. mew：marker 的轮廓线条粗细
  5. markersize：marker的大小

其实这些东西，如果使用 jupyter notebook，都可以直接使用 ?plt.plot来查看。
